# Data

Recommended public datasets for learning and benchmarking:

## Small pedagogical datasets

- Zachary's karate club
- Davis Southern Women
- Florentine families
- Les Misérables co-appearance network
- Political books network
- Football network

## Larger network datasets

- SNAP datasets
- KONECT datasets
- Network Repository
- OpenAlex citation data
- Wikidata-derived networks
- OpenStreetMap-derived spatial networks

## Sociological data cautions

For each dataset, document:

1. What is a node?
2. What is an edge?
3. How were ties collected?
4. Which populations are excluded?
5. Are ties directed, weighted, signed or temporal?
6. What sociological claims can and cannot be made?

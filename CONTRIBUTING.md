# Contributing

This project welcomes additions that combine code, mathematical clarity and sociological interpretation.

## For each new algorithm

Please include:

1. Description
2. Mathematical intuition
3. Sociological interpretation
4. Python implementation
5. Example usage
6. Test
7. References
8. Known limitations

## Suggested structure

```text
network_algorithms/<family>/<algorithm>.py
tests/test_<algorithm>.py
docs/<family>/<algorithm>.md
notebooks/<number>_<algorithm>.ipynb
```

## Style

- Use readable code.
- Add docstrings.
- Explain assumptions.
- Separate prediction from explanation.
- Connect metrics to theory.
